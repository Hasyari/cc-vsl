## GET Specified Data News

Endpoint : GET /news/1

Request Body :

```json
{
  "data": {
    "id": 1,
    "judul": "testing",
    "deskripsi": "testing",
    "image_url": "https://storage.googleapis.com/bucket-544e/1702166501667_8395-pepe-cool.png",
    "createdAt": "2023-12-10T00:01:41.000Z",
    "updatedAt": "2023-12-10T00:01:41.000Z"
  }
}
```
